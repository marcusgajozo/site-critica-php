## Projeto de Programação Web
### Site Crítica de Filmes

Bem-vindos ao site de crítica de filmes! Aqui você pode encontrar avaliações honestas e opiniões sobre os lançamentos mais recentes.

Entre as funcionalidades oferecidas pelo site, destacamos o cadastro de usuários, que permite que você crie uma conta exclusiva e tenha acesso a todas as funcionalidades disponíveis. Além disso, apenas administradores podem cadastrar filmes no banco de dados do site, garantindo assim a qualidade das informações.

Para deixar sua opinião e compartilhar seus pensamentos com outros usuários, adicionamos uma funcionalidade de comentários nas páginas dos filmes. E, para facilitar a busca por filmes específicos, incluímos uma funcionalidade de busca no site.

Por fim, o design responsivo do nosso site oferece uma experiência suave e completa tanto em dispositivos móveis quanto em desktops.

Esperamos que aproveitem as críticas e fiquem à vontade para entrar em contato conosco se tiverem alguma sugestão ou dúvida. Obrigado por visitar!
